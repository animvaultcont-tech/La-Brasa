# La Brasa — Restaurant & Café Website

A modern, luxury-editorial static website for a fictional restaurant & café. Built to showcase freelance web development skills with clean HTML, CSS, and vanilla JavaScript.

---

## 🚀 Features

- **Fully Responsive** — Works perfectly on desktop, tablet, and mobile
- **Custom Cursor** — Gold dot cursor with smooth follower animation (desktop only)
- **Preloader** — Elegant branded loading screen
- **Animated Navbar** — Hides on scroll down, reveals on scroll up, blurs on scroll
- **Scroll Reveal Animations** — Elements fade and slide in as you scroll
- **Interactive Menu Tabs** — Starters, Mains, Desserts, Drinks — all switchable
- **Auto Testimonial Slider** — Pauses on hover, with dot indicators
- **Reservation Form** — Full validation, simulated submission with success message
- **Parallax Hero** — Subtle parallax effect on the hero background
- **Number Counter Animation** — Stats animate when scrolled into view
- **Gallery Grid** — Hover labels on gallery items
- **Mobile Menu** — Animated hamburger with full-screen overlay menu
- **SEO Ready** — Meta tags, Open Graph, sitemap.xml included

## 🛠️ How to Run Locally

1. **Unzip** the folder
2. Open `index.html` in any modern browser (Chrome, Firefox, Edge, Safari)
3. Or use VS Code **Live Server** extension for best experience

> ✅ No build tools, no npm, no dependencies — pure HTML/CSS/JS

---

## 🌐 How to Deploy on GitHub Pages

1. Create a new GitHub repository
2. Upload all files (keep folder structure)
3. Go to **Settings → Pages**
4. Set source to **main branch, root folder**
5. Your site will be live at `https://yourusername.github.io/repository-name`
